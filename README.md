# Cycle-GAN-PyTorch
PyTorch implementation of the Cycle GAN paper.







## Dependencies

- imageio == 2.9.0
- numpy == 1.19.2
- opencv_contrib_python == 4.4.0.44
- torch == 1.6.0
- torchsummary == 1.5.1
- tqdm == 4.50.0

## Installation

```bash
pip3 install -r requirements.txt
```
